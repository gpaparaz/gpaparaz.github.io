The Mixologist's Playground

The Mixologist's Playground è un progetto realizzato nell'ambito dell'esame Applicazioni Web: Progettazione e Sviluppo dell'Università degli Studi di Milano-Bicocca.
Il progetto consiste nel realizzare un sito web in React.js che utilizzi come fonte di dati delle API open source e la lettura di dati da un json locale. Nello specifico per questo progetto ho utilizzato le API di TheCocktailDB.
Il tema che ho scelto è quello di un cocktail bar che serve principalmente cocktail e come accompagnamento cicchetti e varie. Tramite questo sito è possibile navigare nel database dei cocktail sfruttando alcuni filtri offerti dai creatori delle API; è possibile inoltre visualizzare i dettagli del singolo cocktail per poterlo ricreare.
